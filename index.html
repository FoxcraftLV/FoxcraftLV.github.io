Bienvenue dans le Wiki du Modding de Mindustry

Ce wiki contient toutes les informations nécessaires pour modder le jeu Mindustry et créer des modifications personnalisées grâce à notre outil facile à utiliser.

Vous y trouverez des guides détaillés, des exemples de code, ainsi que des ressources pour commencer vos projets de modding.

Installation de l'outil de modding

Suivez les étapes ci-dessous pour installer l'outil nécessaire au modding de Mindustry :

  1. Téléchargez le kit de modding depuis GitHub.
  2. Extrayez le fichier téléchargé dans un répertoire de votre choix.
  3. Assurez-vous d'avoir une version de Java compatible installée sur votre machine.
  4. Ouvrez l'outil de modding et commencez à créer vos modifications !

API de Mindustry pour le Modding

L'API de Mindustry vous permet de manipuler les éléments du jeu comme les blocs, les unités, et les ressources. Voici un exemple simple :

                    // Exemple de code pour créer un nouveau bloc
                    const newBlock = new Block("myCustomBlock", {
                        size: 2,
                        health: 100,
                        buildSpeed: 0.5
                    });
                    newBlock.add();

Pour plus d'informations, consultez la documentation complète de l'API.
